Detailed Description

template<typename ObjectType>
struct limes::ds::LinkedListModel< ObjectType >

This class is an abstract model of a node in a linked list.

This class does not contain the data it references, it is designed to be insertable into any other predefined class to make the class a node of a linked list. For example:

struct MyStruct
{
int someData;
double someOtherData;
limes::ds::LinkedListModel<MyStruct> listNode { *this };
};

Internally, this class simply holds pointers to the next and previous nodes of the list.
