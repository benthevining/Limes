Detailed Description

template<typename ObjectType, size_t Rows, size_t Columns>
class math::Matrix< ObjectType, Rows, Columns >

This class is a matrix, a rectangular array of objects arranged in rows and columns.

Matrices have a compile-time defined size.

Template Parameters
ObjectTypeThe type of objects for the matrix to hold
RowsThe number of rows for the matrix to contain
ColumnsThe number of columns for the matrix to contain
