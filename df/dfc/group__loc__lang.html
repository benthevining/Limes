LIMES_EXPORT Language limes::locale::getLanguage ( const std::string_view &  languageName)

#include <languages.h>

Returns the Language object representing the language with the given name.

If the passed language name is empty, an assertion is thrown. A null Language object may be returned if no known language matches the given language name; you should check isValid() on the returned Language object before using it.

◆ getLanguageForCode()

LIMES_EXPORT Language limes::locale::getLanguageForCode ( const std::string_view &  ISOlanguageCode)

#include <languages.h>

Returns the Language object representing the given ISO language code.

If the given language code is 2 characters, it is interpreted as an ISO 639-1 code. If it is 3 characters, it's interpreted as an ISO 639-2 code. Otherwise, an assertion is thrown. A null Language object may be returned if no known language matches the given ISO language code; you should check isValid() on the returned Language object before using it.

See also
getLanguage()
