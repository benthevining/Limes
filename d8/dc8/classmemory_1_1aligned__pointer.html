Detailed Description

template<typename T, bool UseExceptions = true>
class memory::aligned_pointer< T, UseExceptions >

An RAII object that calls allocate_aligned() on construction and deallocate_aligned() on destruction.

Template Parameters
TThe type of object that will be owned
UseExceptionsWhether allocation failure should throw an exception
