constexprnoexcept

Executes a potentially-throwing function within a try-catch block, and returns false if an exception is thrown and true if no exceptions are thrown.

This is basically a simple Lippincott function. If the return type of the function is bool, then the value it returns will be returned by try_call if the function does not throw an exception.
