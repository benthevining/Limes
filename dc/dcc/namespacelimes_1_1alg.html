Calls a function with corresponding pairs of elements from two containers.

The function's first two arguments must be the element types of Container1 and Container2 ; if the function accepts a third integer argument, then it will also receive the current index. The iteration stops when the end of the smallest container is reached.

Returns
The number of times the function was called

This function allows structured bindings of iterators, similar to Python's enumerate() method.

Example usage:

std::vector<int> v {7,14,21};
for (const auto [idx, val] : enumerate(v))
