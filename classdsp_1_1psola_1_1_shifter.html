Detailed Description

template<Sample SampleType>
class dsp::psola::Shifter< SampleType >

A class that repitches a stream of monophonic audio using PSOLA-like techniques. This class is essentially a "client" of an Analyzer object, which allows multiple Shifters to be used simultaneously without requiring the expensive analysis process to be repeated.
