Detailed Description

A base class representing an object that calculates a hash function.

You should call the update() method with any data that needs to be hashed, and then call getHash() to finalize the internal calculations and retrieve the final hash value. You shouldn't call update() again after calling getHash() .

See also
MD5, SHA1, SHA224, SHA256, SHA384, SHA512
