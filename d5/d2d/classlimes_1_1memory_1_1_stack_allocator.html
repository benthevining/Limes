Detailed Description

template<class T, std::size_t N, std::size_t Align = alignof (std::max_align_t)>
class limes::memory::StackAllocator< T, N, Align >

An STL-style allocator that uses stack space to allocate objects.

Once the reserved stack space runs out, this allocator falls back to using new calls.

Template Parameters
TThe type of objects to be allocated
NThe number of bytes to reserve on the stack
AlignThe alignment to use
