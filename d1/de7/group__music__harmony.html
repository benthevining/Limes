◆ getFlatsOrder()

consteval std::array<PitchClass, 7> limes::music::getFlatsOrder ( )
noexcept

#include <PitchUtils.h>

Returns an array of pitch class objects representing each flat in the order they appear in key signatures – B, E, A, D, G, C, F.

Each pitch class represents the lowered pitch class – so the first two pitch classes in the returned array are 10 and 3, for Bb and Eb , respectively.

◆ lowestNoteOfMidiOctave()

constexpr int limes::music::lowestNoteOfMidiOctave ( int  octaveNumber)
constexprnoexcept

#include <PitchUtils.h>

Returns the lowest note of a given MIDI octave number (that octave's C key).

The formula is \( note=(O+1)*12 \). This formula puts 60 as the lowest note of octave 4.

Here are the MIDI octaves, as defined by this formula:

Octave number Note number range Pitch range
0 12 - 23 C0 - B0
1 24 - 35 C1 - B1
2 36 - 47 C2 - B2
3 48 - 59 C3 - B3
4 60 - 71 C4 - B4
5 72 - 83 C5 - B5
6 84 - 95 C6 - B6
7 96 - 107 C7 - B7

◆ octaveNumberOfMidiNote()

constexpr int limes::music::octaveNumberOfMidiNote ( int  midiNote)
constexprnoexcept

#include <PitchUtils.h>

Returns the octave number of a given MIDI pitch.

The formula is \( O=\lfloor\frac{note}{12}-1\rfloor \). This formula puts note 69 in octave 4.

Here are the MIDI octaves, as defined by this formula:
