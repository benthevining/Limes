# BinaryBuilder

A command line utility to embed resource files into generated C++ source code.

Usage:

```
BinaryBuilder [-d <outputDirectory> -o <fileBaseName> -ns <nameSpace>] <inputFiles...>
```

## Dependencies

* [limes_core](../../libs/limes_core/README.md)

## Targets

* Limes::BinaryBuilder
