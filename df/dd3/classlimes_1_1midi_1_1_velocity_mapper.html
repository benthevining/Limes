Detailed Description

This class allows you to impose a "sensitivity" parameter on MIDI messages velocities.

If the sensitivity is set to full (100), then the velocity is not changed at all. If the sensitivity is set to off (0), then the velocity is always output as 127, regardless of what the input velocity was.
