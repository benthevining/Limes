◆ getDuration()

constexpr double limes::music::NoteDuration::getDuration ( ) const
constexprnoexcept

Returns the percentage of a whole note represented by this object.

1 represents a whole note, 2 represents a half note, 4 is a quarter note, 8 is an eighth note, and so on. This initial integer value is scaled according to the number of dots this duration has, and returned as a percentage of a whole note. A whole note will return 1, a half note 0.5, a quarter note 0.25, etc.

◆ operator++()

constexpr NoteDuration& limes::music::NoteDuration::operator++ ( )
constexprnoexcept

Increments the duration value to the next shortest notehead kind.

Whole notes are incremented to half notes, half notes to quarter notes, quarter notes to eighth notes, etc. The number of dots is unaffected.

◆ operator--()

constexpr NoteDuration& limes::music::NoteDuration::operator-- ( )
constexprnoexcept

Decrements the duration value to the next longest notehead kind.

Half notes are decremented to whole notes, quarter notes to half notes, eighth notes to quarter notes, etc. Decrementing a whole note has no effect. The number of dots is unaffected.
