Detailed Description

A struct that represents a pitch class.

This class is agnostic to the octave the pitch class appears in; and also to enharmonic spellings – thus, C-sharp and D-flat are considered the same pitch class. A pitch class is represented as an integer between 0 and 11, with 0 being C, 1 being C-sharp/D-flat, and 11 being B natural.

Here are all possible pitch classes:

Integer value Note name
0 C
1 C-sharp / D-flat
2 D
3 D-sharp / E-flat
4 E
5 F
6 F-sharp / G-flat
7 G
8 G-sharp / A-flat
9 A
10 A-sharp / B-flat
11 B
