◆ follow()

FilesystemEntry files::SymLink::follow ( bool  recurse = true) const
noexcept

Follows the symbolic link to find its target.

If the recurse option is true, and the immediate target of this symlink is another symlink, then this function will follow that symlink, and so on, until a non-symbolic-link filesystem object is reached.

Attention
Be careful, there is no protection against cycles in this function, so you may invoke an infinite loop!
