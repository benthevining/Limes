Detailed Description

template<typename ObjectType, size_t totalNumThreads = 2>
class threads::ThreadSafeObject< ObjectType, totalNumThreads >

A wrapper class that makes an object thread safe.

Internally, this class holds multiple copies of the owned object, one for each thread, and updates the other threads' copies any time changes are made. Although this class guarantees thread safety, this class does not guarantee realtime safety.

Template Parameters
ObjectTypeThe type of object to be owned
totalNumThreadsThe total number of threads that will be accessing the object. This is the number of copies of the object that will be maintained internally, and must be greater than 0.
