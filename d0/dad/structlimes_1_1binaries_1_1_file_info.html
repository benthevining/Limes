Detailed Description

This class is an interface to any kind of binary data embedded into your source code.

To embed some external data as a source file, create a static extern array of FileInfo objects initialized with the filename, extension, and raw data of any binary files you wish to embed. Then you can create a free function returning an instance of the FileInfoList class which provides access to the array of FileInfo objects you've embedded in your code. A full example of how to do this can be seen in the BinaryBuilder source code.
