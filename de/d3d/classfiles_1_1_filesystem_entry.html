Detailed Description

The base class for any kind of object in the filesystem.

This class is essentially a wrapper around std::filesystem::path .

See also
File, Directory, SymLink

◆ sizeInBytes()

virtual std::uintmax_t files::FilesystemEntry::sizeInBytes ( ) const
virtual

Returns the size, in bytes, of the filesystem entry referred to by this object.

If the filesystem entry doesn't exist, this will return 0. For directories, this returns the cumulative size of all the directory's contents, recursively.

Reimplemented in files::Directory.
